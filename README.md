# 공인인증서 관련 예제
하드디스크/이동식디스크에서 공인인증서을 읽고, 개인키를 복호화하고, 주민등록번호(및 사업자등록번호 같은 본인식별번호)로 본인확인하는 예제입니다.
한번 해보고 싶었어요.
